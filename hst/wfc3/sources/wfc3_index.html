WFC3 Reference File Images & Tables

(Reference file work by the team is outlined in the WFC3 wiki page.)

Reference files used by the WFC3 calibration pipeline (calwf3 and PyDrizzle/MultiDrizzle) are updated on a regular basis. Current reference files files can be retrieved from the HST archive, or downloaded directly from the iref directory. If you choose to access the files via this ftp server, keep in mind that there is a known bug in trying to access files via ftp server using Safari. The recommended browser is Firefox.

If you would like to receive an e-mail every time a new WFC3 reference file is delivered and the database is updated, you can subscribe to the wfc3_reffiles_upd mailing list. To subscribe to this list, send an e-mail to MajorDomo @ stsci.edu with the following in the body of the message:
subscribe wfc3_reffiles_upd your_address .
