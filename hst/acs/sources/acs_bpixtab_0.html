ACS Bad Pixel Table

These tables are used by the DQCORR step in CALACS to populate the data quality (DQ) arrays of ACS images with flags for known (quasi-permanent) bad pixels. See the flag definitions for more details. The correct row in the table is selected by DETECTOR, CCDCHIP, CCDAMP, CCDGAIN.
