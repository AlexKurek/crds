ACS CCD Table

These tables define the key characteristics of the ACS CCDs (WFC and HRC) for the CALACS pipeline. They convert the commanded values to calibrated values for each CCD amp. The proper table row is selected by DETECTOR, CCDCHIP, CCDAMP and CCDGAIN.
