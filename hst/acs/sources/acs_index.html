ACS Reference File Tables

Reference files used by the ACS calibration pipeline (CALACS and PyDrizzle/MultiDrizzle) are updated on a regular basis. Current reference files files can be retrieved from the HST archive, or downloaded directly from the jref directory. If you choose to access the files via this ftp server, keep in mind that there is a known bug in trying to access files via ftp server using Safari. The recommended browser is Firefox.

If you would like to receive an e-mail every time a new ACS reference file is delivered and the database is updated, you can subscribe to the acs_reffiles_upd mailing list. To subscribe to this list, send an e-mail to MajorDomo @ stsci.edu with the following in the body of the message:
subscribe acs_reffiles_upd your_address .

See the logsheets for a detailed history of all ACS reference file deliveries and installations up through September 14, 2006. The pixels affected by these reference files are generally identified by data quality flags in the [DQ] extension of calibrated FITS data.
